Отчет
```sh
git loc
```
Краткий отчет
```sh
git loc --oneline
```

Отмотка назад; т.е. возвращает тебя на номер хэша
```sh
git checkout "Хэш"
```

Возвращение из отмотки назад; к самому последнему состоянию
```sh
git checkout master
```

