# Подсказка по GIT

Создание репозитория:
```sh
git init
```
Индексирование:
```sh
git add <namefIle>
```

Сохранение индексированного:
```sh
git commit -m "Название"
```

История работы в Гите:
```sh
git log
```
История работы в Гите (упрощенный обзор)
```sh
git log --oneline
```

Возвращение к указанному месту:
```sh
git checkout "<name file>"
```
